## Call of Burger

- a simple running game built with Java for university course
- I use multiple screenshot of roads to give a feelings of video game

### To play

  - Download or Clone
  - Unzip file
  - Open project in Java Ide
  - Run Main.java
  - Use keyboard left-right-up-down keys
